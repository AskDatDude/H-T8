    Getting Started With Hacking


    Welcome to the exciting and dynamic world of hacking! This guide is designed to provide a practical and focused introduction, ensuring you can confidently take your first steps into ethical hacking. We’ll guide you through setting up your environment, mastering the basics of Kali Linux, and practicing your newfound skills with hands-on challenges. Let’s embark on this journey together, one step at a time.

    Step 1: Setting Up Your Environment

    Install Virtualization Software

    To safely practice hacking, you need a secure and isolated environment. Virtualization software allows you to run multiple operating systems simultaneously on your computer. Here are your options:

    • VMware: Now available for free, VMware is the preferred choice for its better performance and user-friendly interface. It’s an excellent option for running virtual machines.
    • VirtualBox: A free and open-source alternative that’s also reliable for creating virtual environments. Little bit easier to setup.


    Installation Instructions for VMware or VirtualBox:

    • Download VMware or VirtualBox from their official websites.
    • Follow the installation instructions for your operating system (Windows, macOS, or Linux). Here you can find a helpful tutorial for installing VMware: Youtube tutorial
    • Verify the installation by launching the software.

    Install Kali Linux on Your Virtual Machine:

    Kali Linux is a specialized Linux distribution designed for penetration testing and ethical hacking. Here’s how to set it up:

    • Download Kali Linux ISO image from the official website. Choose the version compatible with your system (VMware image or ISO file).
    • Launch your virtualization software and create a new virtual machine:
      • Allocate at least 2 GB of RAM and 20 GB of disk space.
      • Attach the Kali Linux ISO file or VMware image as instructed.
      • (You can change these settings later as well so no stress here)
    • Follow the installation prompts, selecting default options unless you have specific preferences.
    • Start Kali Linux and log in with the default username and password.

    Configure the Environment

    After installing Kali Linux, configure it to optimize performance and usability:

    (The dollar sign is not part of the command, it represents CLI view)

    • Update the System:

      • $ sudo apt update && sudo apt upgrade

      This ensures all packages are up-to-date and secure.

    • Install Guest Additions (VirtualBox): Enable features like clipboard sharing and screen resizing by installing guest additions. Refer to the VirtualBox documentation for detailed steps.
    • Verify Network Connectivity: Test your setup by running:

      • $ ping google.com

      Ensure you can connect to external networks.

    Step 2: Introduction to Kali Linux

    Understanding the Interface

    Kali Linux offers two primary interfaces:

    • Graphical User Interface (GUI): Ideal for beginners but not as powerful as the CLI.
    • Command-Line Interface (CLI): The backbone of Kali Linux. Most tools and scripts are run from the terminal.

    Basic CLI Commands

    Learning the command line is essential for effective navigation and tool usage in Kali Linux. Start with these foundational commands. Each command is accompanied by a description to help you understand its purpose:

    File and Directory Management

    • List files in a directory:

      • $ ls

      Displays the contents of the current directory, showing files and subdirectories. Add flags like -l for detailed information or -a to include hidden files.

    • Change directory:

      • $ cd [directory_name]

      Moves to the specified directory. Use cd .. to go back one level or cd ~ for the home directory.

    • Create a new directory:

      • $ mkdir [directory_name]

      Creates a new directory with the specified name.


    System Monitoring

    • View system information:

      • $ uname -a

      Displays detailed information about the system, including the kernel version and architecture. The -a flag gives all information.

    • Check running processes:

      • $ top

      Displays real-time information about system processes, memory usage, and CPU activity.

    • View disk usage:

      • $ df -h

      Shows the amount of disk space used and available on your file system. The -h flag presents the information in a human-readable format (e.g., GB, MB).


    Networking Basics

    • Check network interfaces:

      • $ ifconfig

      Displays information about network interfaces, including IP addresses and MAC addresses. Useful for diagnosing network issues or checking connection settings.

    • Check network connectivity:

      • $ ping [website]

      Tests network connectivity by sending packets to a specific website or IP address. Press Ctrl + C to stop the process.

    • Scan for open ports:

      • !!!

      CAREFUL HERE NOW! Port scanning without permission is illegal and can lead into serious consequences. Always ensure you have permission before scanning any network or system.

      !!!

      $ nmap [IP_address]

      Performs a network scan on the specified target IP address to discover open ports, services, and potential vulnerabilities. Use the -p flag to specify a port range.


    Explore Command Documentation

    For a deeper understanding of these commands and their options, use the manual (man) pages:

    $ man [command_name]

    For example, to learn about ls, run:

    $ man ls

    This displays detailed information about the command, including its syntax, options, and usage examples.


    Exploring Pre-installed Tools

    Kali Linux is preloaded with powerful tools. Start with these three:

    • Nmap: A network scanning tool for discovering hosts and services on a network. Use it to identify open ports, detect vulnerabilities, and map network topology.
    • Metasploit: A penetration testing framework that helps you find, exploit, and validate vulnerabilities in a system. It’s a powerful tool for ethical hacking and security testing.
    • Wireshark: A network protocol analyzer that captures and inspects data packets in real-time. Use it to troubleshoot network issues, analyze traffic, and detect security threats.

    Run the tools in the terminal and explore their help documentation by appending --help to commands.

    Step 3: Practicing Your Skills

    Beginner-Friendly Challenges

    Once you’ve set up Kali Linux and practiced basic commands, enhance your skills with these platforms:


    OverTheWire: Bandit

    • Visit OverTheWire and start with the Bandit wargame.

      • This challenge teaches Linux basics, including file manipulation and command usage, in an engaging and progressive format.


    Hack The Box: Starting Point

    • Join Hack The Box to access a wide range of realistic hacking challenges.

      • Begin with the Starting Point challenges, which simulate real-world scenarios for penetration testing.

      The platform offers machines, challenges, and labs to test your skills in a safe and controlled environment.


    TryHackMe: Linux Fundamentals

    • Explore TryHackMe to learn Linux fundamentals and security concepts.

      • The platform offers guided tutorials, hands-on labs, and challenges to help you build practical skills in ethical hacking.

    Step 4: Building a Learning Routine

    Daily Practice Plan

    • Dedicate 1 hour weekly to exploring a new tool in Kali Linux.
    • Solve at least one challenge or lab on OverTheWire, TryHackMe, or Hack The Box weekly.

    Be realistic with your goals, don't make them too extreme and try to stick with your plan long term


    Document Your Progress

    Maintain a log of:

    • Commands and tools you’ve learned.
    • Challenges you’ve completed and key takeaways.
    • Areas where you need more practice.

    Make a system that works for you, whether it’s a physical notebook, digital document, or blog. Also try to be organized and keep your files, notes and logs in order.




    By following this expanded guide, you’ll build a strong foundation in ethical hacking. Remember, consistent practice and curiosity are your greatest allies. Take it step by step, stay ethical, and enjoy the journey. Happy hacking!
